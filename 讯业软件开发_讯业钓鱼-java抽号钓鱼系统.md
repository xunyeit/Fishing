讯业软件开发/讯业钓鱼-java抽号钓鱼系统

### 一、产品概述
**“****钓鱼****”是一款专为垂钓爱好者打造的****一站式智能钓鱼服务平台****，致力于连接钓友与优质钓场资源，提供从****场地预约、导航出行、在线点单、计时管理到回鱼结算****的全流程数字化服务。平台整合全国范围内的专业钓鱼基地、休闲渔场、生态养殖园等资源，支持用户在线查看实时信息（如主养鱼种、放鱼量、人气热度）、预约钓位、下单购买饵料和餐饮，并通过智能计时与电子结算系统提升钓鱼体验效率。**

**该应用面向广大钓鱼爱好者、家庭亲子出游群体及户外运动爱好者，结合移动端便捷操作与线下实体服务，打造“线上预订、线下畅享、数据可查”的智慧垂钓新生态。无论是新手入门还是资深钓手，都能在平台上找到适合自己的钓场与服务，让每一次垂钓都更省心、更高效、更有成就感。**

### 二、核心功能
1. **导航出发****  
****根据用户所在位置，自动规划最优路线并跳转至地图导航软件，确保顺利抵达目标钓场。**
2. **联系钓场****  
****提供钓场联系电话或客服入口，支持用户提前咨询开放时间、天气情况、是否需要预约等信息。**
3. **钓场微信****  
****快速进入钓场官方微信群，获取实时通知、活动公告、群内交流等社交互动内容。**
4. **计时****  
****开启钓鱼计时器，记录实际作钓时长，用于后续费用结算或个人成绩统计。**
5. **点单****  
****在线订购饵料、饮料、餐食、烧烤套餐等配套服务，支持现场扫码支付，减少排队等待。**
6. **购物车****  
****将所需商品加入购物车，统一结算，提升多品类消费的便捷性。**
7. **点击回鱼****  
****捕获鱼类后，通过App一键提交“回鱼”记录，系统自动登记捕获数量与重量，简化人工核验流程。**
8. **回鱼金额****  
****实时显示当前回鱼累计金额，便于用户掌握收益情况，尤其适用于“按斤计价”的模式。**
9. **回鱼次数****  
****记录当天成功回鱼的总次数，可用于竞技比赛或个人挑战目标追踪。**
10. **回鱼数量****  
****统计当日捕获鱼只总数，支持按种类分类展示，帮助用户分析作钓效果。**
11. **我的订单****  
****查看所有已创建的钓鱼订单，包括待使用、已完成、已取消等状态，支持订单详情查询与售后服务申请。**
12. **消费记录****  
****展示每次消费明细，包括门票、饵料、餐饮、服务费等，实现财务透明化管理。**
13. **投诉建议****  
****提交对钓场服务质量、设施设备、工作人员态度等方面的反馈，平台将及时处理并优化体验。**
14. **钓位预约****  
****（主界面功能）选择具体钓场后，可查看小坑位、中坑位、大坑位等不同规格的空闲时段，并完成在线预订。**
15. **玩法与鱼种信息****  
****显示当前钓场的玩法（如正钓、混养）、主要鱼种（如鲤鱼、鲫鱼、草鱼）、放鱼量、票价等关键信息，辅助决策。**
16. **人气热度****  
****展示当前钓场的人气指数（如“2770”），反映其受欢迎程度，帮助用户判断是否拥挤。**
17. **开钓时间****  
****明确标注每日开钓时段（如09:00-18:00），避免误入闭场时间。**
18. **钓场分享****  
****支持将心仪钓场推荐给好友，一键生成链接或海报，促进社交传播与口碑裂变。**
19. **电子凭证****  
****（隐含功能）完成预约后生成电子入场券，扫码即可入园，无需纸质票证。**
20. **数据可视化报表****  
****（可扩展功能）长期使用可生成个人钓鱼报告，包含总钓时、总回鱼量、平均收获率等，助力经验积累。**

### 三、产品价值
#### **用户价值：让钓鱼更轻松、更智能、更有趣**
+ **省时省力****：告别电话询问、现场排队，一键预约、导航直达，大幅提升出行效率。**
+ **信息透明****：实时了解钓场鱼情、放鱼量、价格、人气等信息，避免“踩雷”或“扑空”。**
+ **服务便捷****：在线点餐、扫码付款、回鱼登记，减少繁琐流程，享受“科技+自然”的融合体验。**
+ **数据记录完整****：自动保存每次钓鱼的时间、地点、成果，形成专属钓鱼档案，便于复盘与分享。**
+ **社交属性强****：通过微信群、回鱼排行榜、挑战赛等功能，增强钓友之间的互动与归属感。**

  
 

## 架构特点
### 架构特性
+ **前后端分离架构，独立开发，符合主流开发模式**
+ **前端以Vue3+Vite为主技术，AntdV为UI界面框架**
+ **后端SpringBoot3为基础，MybatisPlus为数据操作框架，Redis为缓存框架**
+ **Maven多模块管理，插件化开发，方便安装、卸载、升级，降低耦合**
+ **业务模块与API抽离，模块之间便捷引用**
+ **数据库设计精巧，字段规范、易于扩展**
+ **支持国产密码算法加解密，等保测评国产项目无压力**
+ **支持MYSQL、ORACLE、SQLSERVER、PGSQL等主流标准结构式数据库**
+ **支持达梦、人大金仓、南大通用、九有、瀚高、虚谷数据库等国产数据库**
+ **支持中创、宝蓝德、东方通等中间件**
+ **支持Windows、Linux操作系统、国产操作系统部署**

### 功能特性
+ **完善的系统基础功能，满足使用需求，避免重复造轮子**
+ **支持本地文件、阿里云文件、腾讯云文件、MINIO文件上传**
+ **支持本地邮件、阿里云邮件、腾讯云邮件发送**
+ **支持阿里云短信、腾讯云短信发送**
+ **B、C端双账号认证体系，会话治理各自独立**
+ **完善的登录日志、操作日志、异常日志等审计功能**
+ **完善的会话监控、数据源监控、系统监控等必备监控功能**
+ **支持组织机构、权限管理、定时任务、系统配置等基础功能**

### 安全特性
+ **采用SaToken轻量级 Java 权限认证框架，功能强大、学习成本低**
+ **支持登录认证、权限认证、单点登录、三方登录、OAuth2.0等认证模式**
+ **增强的RBAC权限设计，资源于接口独立授权，更加灵活**
+ **支持按钮级别细粒度独立授权，界面按钮动态展示**
+ **支持API接口注解式、路由拦截式鉴权，防止越界访问**
+ **独创的数据范围机制，每个接口都可以配置不同数据范围**
+ **支持限流防抖，防重复提交，有效阻止脏数据产生**
+ **密码、手机、身份证号等使用国密算法加密传输、加密存储，数据更安全**
+ **操作日志使用SM2进行完整性保护，满足安全审计要求**

### 界面特性
+ **Vue3 + Vite为基础，AntdV为界面UI框架，视觉风格清新简洁**
+ **精细化设计，注重界面的每一处细节，操作轻松友好**
+ **暗黑风格、经典菜单、双排菜单、多页签、目录坞、主题切换等功能应有尽有**
+ **统一的网络框架、API接口拦截框架，拿来即可上手**

+ **Node.js 14x**

+ **阿里云 OSS – 可更换其它。**
